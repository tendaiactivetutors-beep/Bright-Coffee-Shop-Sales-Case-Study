# ☕ Bright Coffee Shop Sales Analysis Jan- June 2023

## 📌 Project Overview

This project analyzes transactional sales data from *Bright Coffee Shop* to generate actionable business insights for a newly appointed CEO.

The goal is to identify revenue drivers, customer behavior patterns, and opportunities to improve overall sales performance using SQL, data processing, and dashboard visualization.

---

## 🎯 Objectives

The analysis focuses on answering key business questions:

* Which products generate the most revenue?
* What time of day does the store perform best?
* What are the sales trends over time?
* Which products are underperforming?
* How can the business improve sales performance?

---

## 🔄 Data Processing

### Key Transformations

* Cleaned `unit_price` (converted comma values to decimals)
* Created calculated field:

```
total_amount = unit_price × transaction_qty
```

* Structured dataset for analysis and dashboarding

### Process Flow

1. Raw Excel dataset
2. Converted to CSV
3. Loaded into Databricks / SQL environment
4. Cleaned and transformed data
5. Exported for dashboard creation

---

## 📊 Key Metrics (KPIs)

* **Total Revenue** → Overall business performance
* **Total Orders** → Transaction volume
* **Average Order Value (AOV)** → Customer spending behavior

---

## 📈 Data Analysis & Insights

### 1. Product Performance

* Coffee products generate the highest revenue
* A small number of products contribute most of the sales 
* Some products consistently underperform

👉 **Business Action:**

* Increase stock of best-selling items
* Promote or bundle low-performing products

---

### 2. Time-Based Analysis

* Peak sales occur during morning hours (breakfast rush)
* Certain days show significantly higher revenue

👉 **Business Action:**

* Allocate more staff during peak hours
* Run promotions during slow periods

---

### 3. Sales Trends

* Daily revenue trends show fluctuations that may relate to demand patterns

👉 **Business Action:**

* Monitor trends to forecast demand
* Plan inventory accordingly

---

### 4. Customer Behavior

* Customers tend to purchase frequently but with moderate order values

👉 **Business Action:**

* Introduce upselling strategies (combos, add-ons)
* Implement loyalty programs

---

## 📊 Dashboard Overview

The dashboard includes:

* KPI cards (Revenue, Orders, AOV)

## 🧠 Key Business Recommendations

* ⏰ Optimize staffing based on peak hours

* 📦 Improve inventory planning using sales trends
* 💡 Introduce loyalty programs to increase repeat purchases
